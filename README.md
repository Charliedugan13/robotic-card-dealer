# Robotic-Card-Dealer
An automated card-dealing machine that accepts user input for number of players and cards per player, then distributes cards evenly using a rotating mechanical system.
## Hardware
- Microcontroller
- Stepper motor
- DC motor + wheel
- Motor driver
- 16x2 LCD
- Push buttons

## Software
- Language: C++
- Libraries: Adafruit Motor Shield, LiquidCrystal I2C

## How It Works
1. User enters number of players/number of cards into an LCD display controlled by push buttons
3. System calculates equal angular spacing 
4. Stepper motor rotates to each position
5. DC motor releases one card per player per round
